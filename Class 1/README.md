# Class 1 - The "Don't Go Broke" Setup & First Computer


## Topics Covered:

- **Setting Up Google Cloud Account:** The most important thing in using a cloud provider is to have an account with that provider. This section will give you guidance on how to create your Google Cloud Account
- **Setting Up Billing Alert:** The most common trap in cloud computing is to go over-budget. This section will guide you how to setup alerts on how much of budget is consumed. These alerts will help you to monitor and stay within your budget
- **Creating First Compute Engine:** Everything in cloud is just a architecture of Virtual Machines. This section will guide you how to start using cloud computing by creating your first Virtual Machine(or Compute Engine in case of GCP)
